SAP Java IDoc Library 3.1
Installation
 
General information
The SAP Java IDoc Library (JIDocLib) 3.1 requires Java Runtime Environment (JRE) version 8 or 11.
The SAP Java IDoc Library 3.1 is an add-on library for the SAP Java Connector 3.1 (currently at least version 3.1.5 is required) and cannot be run separately. It offers a set of additional interfaces and classes which allow sending, receiving and processing IDocs in a convenient way.
 
Archive
The SAP Java IDoc Library 3.1 is packaged in the archive sapjidoc31P_2-10009485.zip, which includes the sapidoc3.jar, the JavaDoc for the library and examples for both an IDoc client and an IDoc server application.
 
Instructions
In order to install the SAP Java IDoc Library 3.1 extract the archive to some folder of your convenience. Optionally, you may copy the sapidoc3.jar file into the same folder where you have put the sapjco3.jar file (see the installation instructions for SAP JCo 3.1).
Then you need to add the sapidoc3.jar archive to your CLASSPATH environment variable or to the classpath argument of the java call that starts your application.
